# Trivia_Game
I am building a command line trivia game in Python that runs locally on your device. It fetches real questions from a free API, shows them to you one at a time with 4 choices, takes your input, and tells you if you're right or wrong. At the end it shows your final score. 

---

# How To Play

1) Download the .zip file
2) Exctract the .zip file
3) Open the extracted .zip file, right click in a blank space and press run in terminal
4) type **python3 game.py** and hit enter inside the terminal.
5) **HAVE FUN!!!**

---

# Another way to play

1. Clone the repo
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it:
   - **Linux/Mac:** `source venv/bin/activate`
   - **Windows:** `venv\Scripts\activate`
4. Install dependencies: `pip install requests rich`
5. Run: `python3 game.py`

## Features
- 10 random trivia questions per game
- Multiple choice answers
- Score tracking
- Local leaderboard saved to a file
